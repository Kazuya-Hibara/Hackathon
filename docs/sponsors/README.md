# スポンサーツール リファレンス

審査基準にスポンサーツール活用度が含まれる。以下を可能な範囲で組み込む。

| Tool | 用途 | 必須度 |
|------|------|--------|
| **Zeabur** | デプロイ・ホスティング | **必須**（デプロイ先） |
| **TiDB Cloud Zero** | DB（MySQL互換、ベクトル検索付き） | 必要に応じて |
| **Agnes AI (ZenMux)** | LLM推論 | 組み込み推奨 |
| **Bright Data** | Web検索・スクレイピング | 組み込み推奨 |
| **Mem9** | 永続メモリ | 組み込み推奨 |
| **Nosana** | 分散GPU推論 | 余裕があれば |
| **QoderWork** | 提出デッキ生成 | 提出時に使用 |

各ツールの詳細は個別ファイルを参照。
